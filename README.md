# Blazor JS call example

Blazor JS调用示例

## JS调用方式 JS call type

### 1. NativeCall

调用JS原生函数

Call native js function


### 2. DynamicFunction 

动态生成JS函数然后调用

Create function dynamically and call


### 3. DotNetInvoke

同步调用.NET方法(只能在wasm模式使用)

Invoke .NET method synchronously (Can only be used in wasm mode)


### 4. DotNetInvokeAsync

异步调用.NET方法

Invoke .NET method asynchronously
